This program prompts the user for a random number seed. Then it simulates a race between a hare, tortoise, and bee.

To compile the program, run the command:
	@gcc -g -Wall animals.o race.o -o theRace -lm

To run the program, run the command:
	./theRace
and then enter a number when prompted

Sources: I brainstormed the logic behind the program with Yara Nabih.
